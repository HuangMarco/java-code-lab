# Datastructre & Algorithm in Java

Java实现的数据结构&算法

🔧环境和工具
------
  1. Java8 & Maven
  2. Idea or Eclipse

 ⚡️每个算法都附带相应的测试用例，部分地方加上了利用生成的随机数进行的性能实际测试。比如排序，我会用Random预先生成一定数量的随机数。并计算出排序所耗费的时间。

 > 做工程，对于算法性能的态度，除了数学本身关于算法时间空间复杂度的证明，还应该去实际测试相关的性能。
